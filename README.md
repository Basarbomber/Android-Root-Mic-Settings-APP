# Android-Root-Mic-Settings-APP
🎙️ Android Audio Mixer Paths Controller (Root)
This power-user tool allows you to search, bind, and individually configure your Android device's microphones by directly modifying the system's internal audio routing. It was specifically built to fix hardware issues—like disabling a broken microphone or switching audio input sources on the fly.

🤖 Built with Google AI Studio (Gemini)
This entire project is a testament to the power of Vibe Coding. The app, including its layout, XML parsing, and integrated terminal interface, was fully generated using Google AI Studio (Gemini) based on just a few descriptive prompts.

🚀 Features
Root-Powered Engine: Grants direct access to system audio configuration files using secure su shell commands.

Smart Mixer Paths Binding: Automatically locates and binds to your device's mixer_paths.xml file.

Deep Search Function: Instantly search through complex audio paths, XML nodes, and microphone configurations.

Microphone Toggling: Easily enable, disable, or reroute individual microphones (perfect for devices with broken secondary or tertiary mics).

Built-in Terminal: An integrated terminal to execute and monitor audio-related root commands in real-time.

⚠️ Disclaimer: This app requires Root access (Magisk, APatch, or KernelSU) and modifies critical system files. Use at your own risk and always keep a backup of your original mixer_paths.xml!
